## 📞 Need Help?

### Common Issues
- **"Chatbot not initialized"** → Add COHERE_API_KEY to Variables
- **"ModuleNotFoundError"** → Check requirements.txt
- **"Vector DB not found"** → Check DB_PATH variable
- **Slow first request** → Normal on free tier (5-10 sec)
